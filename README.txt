== Screencasts Ruby Brasil

Cada pasta criada aqui é uma apresentação, confira os PRONTOS e FAZER.


== Como participar?

1. Gravando uma apresentação pronta.
2. Criando uma apresentação nova.
3. Ajudando uma apresentação em desenvolvimento.


== Como criar uma apresentação?

Instale o CODEX:
  gem install codex
  
Crie a pasta com o tema da apresentação:
  codex tema
  
Para gerar o html use:
  rake all
  
== Como trabalhar com o CODEX:

Dentro de cada apresentação, a pasta slides contém o texto (markdown), é só editar a vontade.
A pasta UI contém os css e imagens.
